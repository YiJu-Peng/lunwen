# 选课系统数据导入说明

本文件夹包含了用于向选课系统数据库添加测试数据的脚本文件。

## 数据库准备

在运行这些脚本之前，请确保您已经：

1. 安装并启动了MySQL服务器
2. 创建了名为"enrollment"的数据库
3. 在MySQL中设置了root用户的密码为"root"（如果您的密码不同，请修改脚本中的密码）

## 文件说明

本文件夹包含以下文件：

- `subject_teacher_data.sql` - 科目和教师数据
- `student_data.sql` - 学生数据
- `add_data.sql` - 课程和选课记录数据
- `import_all_data.bat` - Windows系统上的一键导入脚本
- `import_all_data.sh` - Linux/Mac系统上的一键导入脚本

## 数据说明

这些脚本将添加以下数据：

1. 40门不同专业的课程，包括：
   - 计算机科学（8门）
   - 电子工程（8门）
   - 数学（8门）
   - 物理（8门）
   - 化学（8门）
   
2. 40名对应的教师数据
3. 40门对应的科目数据
4. 20名不同专业的学生数据
5. 8条选课记录

## 使用方法

### Windows系统

1. 打开命令提示符（CMD）或PowerShell
2. 确保您已经安装了MySQL客户端，并且能够通过命令行访问它
3. 导航到包含脚本的目录
4. 直接双击运行`import_all_data.bat`文件，或在命令行输入：
   ```
   import_all_data.bat
   ```

### Linux/Mac系统

1. 打开终端
2. 确保您已经安装了MySQL客户端，并且能够通过命令行访问它
3. 导航到包含脚本的目录
4. 首先授予执行权限：
   ```
   chmod +x import_all_data.sh
   ```
5. 然后运行脚本：
   ```
   ./import_all_data.sh
   ```

## 注意事项

1. 如果您的MySQL用户名和密码不是"root"和"root"，请修改脚本文件中的相关参数
2. 如果您想要单独运行某个SQL文件，可以使用以下命令：
   ```
   mysql -u root -proot enrollment < 文件名.sql
   ```
3. 脚本执行前会尝试删除已存在的表，以避免数据重复
4. 如果执行过程中出现错误，请检查您的MySQL配置和连接
